## 📌 Project Overview

The **Email Slicer** is a simple Python project that extracts the **username** and **domain name** from an email address.

This project demonstrates basic Python concepts such as **string manipulation, indexing, slicing, and user input handling**.

For example:

Input Email

```
john.doe@gmail.com
```

Output

```
Username: john.doe
Domain: gmail.com
```

---

## 🎯 Objective

The objective of this project is to:

* Learn **string manipulation in Python**
* Extract useful information from an email address
* Practice **Python slicing and indexing**
* Understand how input and output work in Python

---

## ⚙️ How the Program Works

1. The program asks the user to **enter an email address**.
2. It removes any extra spaces using `.strip()`.
3. It finds the **@ symbol** in the email.
4. The email is then divided into two parts:

   * **Username** → text before `@`
   * **Domain** → text after `@`
5. The extracted information is displayed to the user.
